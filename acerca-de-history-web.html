Acerca de History Web

History Web es un proyecto académico que transforma la línea del tiempo histórica en una experiencia web clara, accesible y fácil de mantener. Nació como un ejercicio de calidad de software y hoy funciona como base para seguir integrando más eventos, recursos educativos y funcionalidades orientadas a entornos reales.

Visión del proyecto

Nuestro objetivo es que cualquier estudiante o docente pueda recorrer la historia desde el origen del universo hasta la Edad Contemporánea con una interfaz sencilla, ordenada y coherente. Al mismo tiempo, el sitio sirve como laboratorio para aplicar normas, modelos y métricas de calidad de software.

Cómo construimos History Web

  • Definimos una estructura clara de carpetas: assets, eventos, subeventos y calidad.
  • Diseñamos componentes reutilizables: navbar fijo, tarjetas de eventos y footer unificado.
  • Aplicamos buenas prácticas de accesibilidad: contraste, navegación por teclado y textos alternativos.
  • Concentramos estilos en una sola hoja styles.css usando variables de color y tipografía.
  • Validamos el sitio con pruebas manuales y auditorías automatizadas usando Lighthouse.

Hacia dónde va el proyecto

  • Agregar más eventos y subeventos históricos con nuevas ilustraciones e historias breves.
  • Incorporar secciones interactivas para estudiantes (línea del tiempo, retos y preguntas rápidas).
  • Explorar un módulo de inicio de sesión para guardar progreso y favoritos.
  • Seguir midiendo la calidad del sitio con nuevas métricas y reportes de desempeño.
